Movie API Documentation

Endpoints

GET /api/movies - Get all movies

GET /api/movies/:id - Get a single movie by id

POST /api/movies - Create a new movie

PUT /api/movies/:id - Update a movie by id

DELETE /api/movies/:id - Delete a movie by id

Example Usage

Get all movies:

    fetch('/api/movies')
      .then(res => res.json())
      .then(movies => {
        // use movies data
      })

Get movie with id 123:

fetch('/api/movies/123')
  .then(res => res.json())
  .then(movie => {
    // use movie data 
  })

Create a new movie:

fetch('/api/movies', {
  method: 'POST',
  body: JSON.stringify({
    title: 'New Movie',
    rating: 4.5 
  })
})

Update movie with id 123:

fetch('/api/movies/123', {
  method: 'PUT',
  body: JSON.stringify({
    title: 'Updated Movie',
    rating: 4.5
  })
})

Delete movie with id 123:

fetch('/api/movies/123', {
  method: 'DELETE'
})
